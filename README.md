# Simple Calculator 🧮

A lightweight command-line calculator written in C++ that performs basic arithmetic operations.

## Features

- ➕ Addition
- ➖ Subtraction
- ✖️ Multiplication
- ➗ Division

## Getting Started

### Prerequisites

- A C++ compiler (e.g., `g++`, MSVC, or Clang)

### Compilation

```bash
g++ simplecalculator.cpp -o simplecalculator
```

### Running the Program

```bash
./simplecalculator        # Linux/Mac
simplecalculator.exe      # Windows
```

## Usage

1. Run the program in your terminal
2. Enter the first number when prompted
3. Enter the operator (`+`, `-`, `*`, `/`)
4. Enter the second number when prompted
5. The result will be displayed on the screen

### Example

```
Enter first number: 10
Enter operator (+, -, *, /): *
Enter second number: 5
Result: 50
```

## Notes

- Division by zero is not allowed
- The program accepts integer and/or floating-point numbers
